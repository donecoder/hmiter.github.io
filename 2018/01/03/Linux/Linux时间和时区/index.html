Linux时间和时区

2018-01-03
programming
Linux

如果你的 Linux 系统时区配置不正确，必需要手动调整到正确的当地时区。
NTP 对时间的同步处理只计算当地时间与 UTC 时间的偏移量，因此配置一个 NTP 对时间进行同步并不能解决时区不正确的问题。
所以大家在用了国外云计算服务商如 Microsoft Azure 或其它 VPS、虚拟机时，需要注意是否与中国大陆的时区一致。
